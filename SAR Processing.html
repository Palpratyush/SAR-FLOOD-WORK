SAR Flood Processing Workflow

Synthetic Aperture Radar (SAR) based flood detection using Sentinel-1 data processed within Google Earth Engine for all-weather flood monitoring.

🛰 Data Source

  • Sentinel-1 Ground Range Detected (GRD)
  • VV Polarization
  • Pre-event and Post-event SAR imagery

⚙ Processing Steps

  • Speckle noise reduction
  • Radiometric calibration
  • Terrain correction
  • Backscatter difference computation
  • Threshold-based flood classification
  • Binary flood mask generation

📊 Output

  • Flood extent raster layer
  • Inundation classification map
  • Exported GeoTIFF for network analysis

The SAR workflow ensures reliable flood detection even under cloud-covered conditions where optical imagery fails.
