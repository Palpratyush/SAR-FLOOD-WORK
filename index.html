SAR-NAVIS

Cloud-Based SAR Flood Impact and Route Analysis Using Web GIS

Synthetic Aperture Radar – Navigation & Vulnerability Intelligence System

SAR Processing Road Block & Route Analysis

🌍 Project Overview

This project presents an integrated framework that combines satellite-based flood detection, transportation network vulnerability assessment, and web-based visualization to support disaster response planning.

Sentinel-1 SAR data enables flood mapping under all-weather conditions through backscatter analysis of pre- and post-event imagery.

Flood extent is integrated with OpenStreetMap (OSM) road data to identify submerged road segments and evaluate transportation disruption.

A graph-based network model simulates connectivity loss and generates safe alternative routes using shortest-path algorithms.

🎯 Objectives

  • Flood detection using SAR backscatter analysis
  • Identification of submerged road segments
  • Quantification of transportation network disruption
  • Generation of safe alternative routes
  • Development of a Web GIS visualization platform

🛰 Methodology

A. Flood Detection Using SAR

  • Acquisition of pre-event and post-event Sentinel-1 imagery
  • VV polarization backscatter analysis
  • Threshold-based flood classification
  • Binary flood mask generation

B. Road Network Impact Assessment

  • Extraction of OpenStreetMap road network data
  • Spatial intersection of flood and road layers
  • Identification of submerged segments
  • Calculation of affected road length and disruption percentage

C. Network Analysis & Routing

  • Graph construction based on road connectivity
  • Removal of flooded road segments
  • Recalculation of shortest safe routes
  • Comparative route efficiency analysis

📊 Key Outcomes

  • Accurate flood extent mapping
  • Quantified infrastructure impact
  • Transportation network disruption estimation
  • Identification of safe emergency routes
  • Integrated Web GIS visualization dashboard

💡 Significance

This framework demonstrates how remote sensing and spatial network analysis enhance disaster preparedness and response planning.

The SAR-based flood detection integrated with transportation modeling provides a scalable and near real-time solution for emergency mobility management.
